# 轻量开发者模式 v1

打开游戏地址后附加 `?dev=1`，例如 `https://g.anyq.site/?dev=1`。普通地址不加载面板，也不改变联机协议、服务器物理或正式玩家界面。

## 使用范围

这是单人本地调试工具：参数只写入当前浏览器的 `localStorage`，只会传给浏览器内创建的 `GameRoom`。它不会发送到 WebSocket，也不会影响同房间其他玩家。

## 可视化

- 青色框：角色实际碰撞箱。
- 橙色框：方块碰撞体。
- 红色虚线：镜头中线。
- 左侧红线：可见的淘汰警告线；实际判定仍留有可调的恢复余量。

## 参数

- 镜头速度：镜头加速度和速度上限倍率。
- 追赶补偿：落后镜头中线时的百分比追赶倍率。
- 重力：竖直加速度倍率。
- 碰撞宽度/高度：实时修改角色物理碰撞箱，并重新居中 X 偏移。
- 淘汰余量：可见左边缘之外仍允许追回的距离。

所有滑杆会自动保存。`重开单人` 用当前参数重新起跑；`暂停` 和 `慢放` 只控制单人本地时钟；`重置参数` 恢复复刻默认值。

## 配置交接

点击“导出 JSON”得到可分享的配置文件；点击“导入 JSON”可恢复另一台设备导出的文件。文件格式为：

```json
{
  "version": 1,
  "tuning": {
    "cameraSpeedMultiplier": 1,
    "recoveryMultiplier": 1,
    "gravityMultiplier": 1,
    "hitboxWidth": 37,
    "hitboxHeight": 48,
    "eliminationMargin": 60
  }
}
```

数值会在导入时限幅，错误 JSON 不会覆盖已保存的参数。
